## About

**HelloEarth** is a curated reference for developers working with Earth Observation (EO), remote sensing, and geospatial data. It collects the best learning resources, open datasets, libraries, and domain knowledge needed to go from pixel to product — without the GUI.

> Whether you're writing your first `rasterio.open(...)` or operating a satellite tasking pipeline, this guide aims to be your starting map.
